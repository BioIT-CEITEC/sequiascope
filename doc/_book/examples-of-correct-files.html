2 Examples of Correct Files

2.1 Structure for Patient “DZ1601” 

DZ1601/
├── DZ1601.variants.tsv                 # somatic variants
├── DZ1601_germline.variants.tsv         # germline variants  
├── DZ1601.xlsx                         # gene fusions
├── DZ1601_tumor.bam                    # tumor BAM
├── DZ1601_tumor.bam.bai                # tumor BAM index
├── DZ1601_normal.bam                   # normal BAM
├── DZ1601_normal.bam.bai               # normal BAM index
├── DZ1601_chimeric.bam                 # chimeric BAM
├── DZ1601_chimeric.bam.bai             # chimeric BAM index
├── DZ1601_arriba.pdf                   # Arriba report
├── DZ1601_arriba.tsv                   # Arriba results
├── DZ1601_Blood.tsv                    # expression in blood
├── DZ1601_Liver.tsv                    # expression in liver
└── DZ1601_genes_of_interest.tsv        # genes of interest

2.2 Example variants.tsv 

var_name    sample  gene    chromosome  position    ref alt in_library
chr1:12345:A>G  DZ1601  BRCA1   1   12345   A   G   2/10
chr2:67890:C>T  DZ1601  TP53    2   67890   C   T   1/10
