8 Contact and Support

If you encounter problems:

  1. Check this documentation for solutions to common problems
  2. Verify data format according to the examples provided
  3. Contact technical support with problem description and sample data
  4. Attach log files if available

For faster problem resolution, include: - Screenshot of error message - Sample of directory structure - Example of problematic file (first 10 lines)

8.1 Data Upload Process

This module is the entry point of the entire SequiaViz application.
Before uploading data, make sure you have properly formatted input files and directories according to the specifications below.

8.1.1 Step 1: Directory, patient and dataset selection

  1. Select main directory containing data for all patients
  2. Add patient IDs - you can enter several at once separated by commas or new lines
  3. Select analysis types you want to visualize
  4. Set patterns for BAM files (e.g., “tumor”, “normal”, “chimeric”)
  5. Enter tissue names for expression analysis (e.g., “Blood”, “Liver”)

8.1.2 Step 2: Validation and confirmation

  1. Application searches the selected directory and finds all relevant files
  2. Displays overview of found files for each patient
  3. Marks problems using colored icons:
    • 🟢 Green: All required files found
    • 🟠 Orange: Optional files missing (e.g., BAM for IGV)
    • 🔴 Red: Required files missing or too many files
  4. Confirm configuration to continue
