4 Common Issues and Troubleshooting

4.1 Status Icons and Their Meanings

4.1.1 🟢 Green Icons

  • “Files found”: All required files are available
  • “Paired files OK”: BAM and BAI files are complete pairs
  • “Single file OK”: Required file was found

4.1.2 🟠 Orange Icons

  • “Missing BAM or BAI”: BAM file or its index is missing
  • “Missing PDF or TSV”: Incomplete Arriba pair (PDF/TSV)
  • “Optional files not available”: Optional files are not available

4.1.3 🔴 Red Icons

  • “Required file missing”: Required file for analysis is missing
  • “Multiple files found”: More files found than expected

4.2 Common Errors and Solutions

4.2.1 “Required file missing”

Cause: Required file for selected analysis type is missing
Solution: - Check if the file exists in the patient directory - Verify correct file naming (must contain patient ID) - Make sure the file has the correct extension (.tsv, .xlsx)

4.2.2 “Multiple files found”

Cause: More files of the same type found for one patient
Solution: - Remove or rename duplicate files - Make sure each patient has only one file of each type - Check BAM file patterns - they might be too general

4.2.3 “Missing BAM or BAI”

Cause: BAM file doesn’t have corresponding index (.bai)
Solution: - Generate missing .bai files using samtools index - Make sure .bai files have correct naming - For file.bam there must be file.bam.bai or file.bai

4.2.4 “Missing PDF or TSV”

Cause: Incomplete Arriba output
Solution: - Make sure you have both files (.pdf and .tsv) from Arriba analysis - Check naming - both must contain patient ID
