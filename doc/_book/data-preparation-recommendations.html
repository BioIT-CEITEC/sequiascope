3 Data Preparation Recommendations

3.1 Before Upload

  1. Check data completeness - make sure you have all required files
  2. Verify naming - patient ID must be consistent across all files
  3. Generate BAI indexes for all BAM files
  4. Check format - TSV files must have tabs as separators

3.2 Performance Optimization

  1. File size: Recommended maximum BAM file size: 2 GB
  2. Compression: Use compressed formats where possible
  3. Network location: For large files consider local storage

3.3 Backup and Versioning

  1. Backup original data before any modifications
  2. Use versioning to track changes in data
  3. Document changes in data for future reference
